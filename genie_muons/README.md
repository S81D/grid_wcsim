`thru_genie_muons.txt` contains a list of 40k muon events filtered from the GENIE ntuples generated by James Minock (found here: `/pnfs/annie/persistent/simulations/genie3/G1810a0211a/standardv1.0/world`).

Muon events were selected with basic cuts:
  - More than 0 registered PMT hits within the tank
  - Originated in front of the FMV and had an initial direction such that they could hit the MRD
  - Crossed the FMV and the tank

For each event (row), the textfile columns correspond to:

```
vertex_position_x[cm], vertex_position_y[cm], vertex_position_z[cm], normalized_dir_x, normalized_dir_y, normalized_z, kinetic_energy[MeV]
```

where the vertex positions, directions, and energy are all initial conditions as extracted from the GENIE files. The coordinates are relative to the center of the FMV (defined as 0,0,0).

List generated by Johann Martyn.
